# Feature Branching
Mit Feature Branching wird für jedes neue Feature oder Bugfixing ein eigener neuer Branch erstellt. 
Erst wenn alles fertig und getestet ist, wird dann in Main gemerged.

## Ablauf
- Vom Main neuen Branch für Feature oder Bugfix erstellen
- In neuem Branch entwickeln
- Wenn alles fertig ist -> PR erstellen
- Branch in Main mergen
